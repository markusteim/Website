The text on the website can be increased by using CTRL + when using a laptop.
On mobile the text can be increased by changing browser default font size.

Users should be able to change font colors within the browsers.

Navigation landmarks have been used on the website to allow screen reader users understand access content.

Content on most of the pages can be accessed by text-to-speech tools.

All of the images on the website have been provided with a descriptive alt tag.


I am committed to creating a website that can be accessed by any person. This site has been built using code compliant with W3C standards for HTML and CSS. If there are any difficulties using the website please do not hesitate to contact me using the contact form on the Contact page.
